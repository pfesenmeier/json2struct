# Json 2 Struct for golang

Install:

    cargo install json2struct

USAGE:
    json2struct [OPTIONS] <json>

FLAGS:

    -h, --help       Prints help information
    -V, --version    Prints version information

OPTIONS:

    -o <omitempty>        是否添加 omitempty [default: true]

ARGS:

    <json>    Input a json
